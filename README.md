# Breakout peli
Breakout pelissäni ideana on liikuttaa nuolinäppäimillä (vasen ja oikea) lautaa niin, että pallo osuu lautaan ja näinollen kimpoaa takaisin ylöspäin.
Tavoitteena on tuhota kaikki tiilet.
